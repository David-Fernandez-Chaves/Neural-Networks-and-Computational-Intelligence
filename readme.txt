The code is divided into two folders, one for each assignment.
To replicate the values exposed in the report, do the following:

ASSIGMENT 1 <---------------------------

Open the file Assigment 1/Assigment1.m:
-Setting range of Alpha. Line:6. Default: 0.75:0.1:3.5
-Setting number of dimensions. Line:11. Default: 20
-Setting number of experiments. Line:12. Default: 500

Run the file.

BONUS1:
Open the file Assigment 1/Assigment1_Bonnus1.m
-Setting range of Alpha. Line:6. Default: 0.75:0.1:3
-Setting number of dimensions (N). Line:7. Default: 2:1:40
-Setting number of experiments. Line:9. Default: 100

Run the file.

BONUS2:
Open the file Assigment 1/Assigment1.m
-Setting range of Alpha. Line:6. 
	For Dataset l.s. Default: 0.75:1:1
	For Dataset l.n.s. Default: 3.4:1:3.5

-Setting number of dimensions. Line:11. Default: 20
-Setting number of experiments. Line:12. Default: 1
-Uncoment line 33 and coment line 29

Run the file.

BONUS3:
Open the file Assigment 1/Assigment1.m:
-Setting range of Alpha. Line:6. Default: 0.75:0.1:3.5
-Setting number of dimensions. Line:11. Default: 20
-Setting number of experiments. Line:12. Default: 500
-Choose the value of C in the last parameter of the function written in the line: 29. Default: 0.1

BONUS4:
Choose whether you want results with clamped or unclamped by uncommenting or commenting respectively the lines 7 and 10 of the file Assigment 1/Perceptron.m

Then open the file Assigment 1/Assigment1.m
-Setting range of Alpha. Line:6. Default: 0.75:0.1:3.5
-Setting number of dimensions. Line:11. Default: 20
-Setting number of experiments. Line:12. Default: 500

Run the file.


ASSIGMENT 2 <---------------------------

Then open the file Assigment 2/Assigment2.m
-Setting range of Alpha. Line:6. Default: 0.75:0.1:3.5
-Setting number of dimensions. Line:7. Default: 20
-Setting number of experiments. Line:8. Default: 20
-Setting level of noisy. Line 10. Default: 10

Run the file.

To show and hide the results of the perceptron generalization error (for the Bonus of dataset with noise), uncomment and comment respectively the lines: 75, 78 and 79

All BONUSES are resolved with the same instructions by changing these parameters.
